# Titanic Data Cleaning & Preprocessing

## 📌 Internship Task
This project was completed as part of the Data Science Internship Program by Synent Technologies.

## 👨‍💻 Submitted By
Ved Patel

## 📂 Dataset
Titanic Dataset

## 🎯 Objective
The objective of this project is to clean and preprocess the Titanic dataset by handling missing values, removing duplicates, converting data types, renaming columns, and preparing the dataset for further analysis.

## 🛠️ Tasks Performed
- Handled missing values
- Removed duplicate records
- Renamed columns
- Converted data types
- Performed exploratory data analysis
- Created data visualizations

## 📊 Visualizations Included
- Survival Count
- Age Distribution
- Survival by Gender
- Survival by Passenger Class

## 🧰 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📈 Key Insights
- Missing values were successfully handled.
- Cabin column was removed due to excessive missing values.
- Most passengers were between age 20–40.
- Female passengers had higher survival rates.
- The cleaned dataset is now ready for further analysis or machine learning.

## 📁 Output
Cleaned Titanic dataset ready for analysis.
